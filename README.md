# project-react-01

## React Curriculum vitae

Crear tu curriculum vitae en React, analiza la estructura propuesta e inspirate para mostrar tu información.

![react-cv](./screenshot.png)

### Instalación
```
npm install
```

### Server
```
npm run server
```

### Ejecución
```
npm run start
```

### Compilar
```
npm run build
```

### Pruebas Unitarias
```
npm run test
```

### ESlint
```
npm run lint
```
